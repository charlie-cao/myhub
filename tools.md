- htop: 进程管理
    - install: yum install htop
    - 
        `htop` 是一个交互式的进程查看器，用于 Linux 系统。与传统的 `top` 命令相比，`htop` 提供了一个彩色的界面，并支持鼠标操作，使得用户界面更为友好。以下是 `htop` 的基本使用和操作：

        1. **启动**:
        - 在终端中输入 `htop`，然后按回车。

        2. **界面说明**:
        - **顶部**：显示系统概览，包括 CPU 使用率、内存使用、负载平均值和任务数等。
        - **中部**：列出当前正在运行的进程，包括进程 ID、用户、CPU/MEM 使用率、命令名等。
        - **底部**：显示快捷键和相应的操作。

        3. **导航**:
        - **上/下箭头键**：在进程列表中上下移动。
        - **左/右箭头键**：在顶部面板中移动，选择不同的 CPU 或内存条。

        4. **排序**:
        - **F6** 或 **`<` / `>`**：选择一个属性对进程进行排序，例如按 CPU 使用率、内存使用或进程 ID。

        5. **搜索**:
        - **F3**：启动搜索模式。输入您要查找的进程名，`htop` 会自动定位到匹配的进程。

        6. **进程管理**:
        - **F9** 或 **`k`**：杀死选定的进程。
        - **F7**：降低选定进程的优先级。
        - **F8**：提高选定进程的优先级。

        7. **树视图**:
        - **F5**：切换到树视图，可以看到进程之间的父子关系。

        8. **刷新**:
        - **F10** 或 **q**：退出 `htop`。

        9. **其他**:
        - **h** 或 **F1**：显示帮助屏幕，列出所有可用的快捷键和操作说明。

        `htop` 是一个非常有用的工具，特别是当您需要快速查看和管理系统上的进程时。由于其交互性和直观的用户界面，`htop` 是很多 Linux 用户的首选进程查看器。

- tmux：终端复用器，允许在单个终端窗口中多次启动和控制多个终端会话。
    - 简单手册
        `tmux` 是一个终端复用工具，允许你在一个终端窗口中创建多个会话和窗格，以便在同一个终端中同时运行多个命令和任务。这有助于提高终端的效率，特别是当你需要在服务器上进行远程工作时。以下是 `tmux` 的基本用法：

        1. **安装和启动 `tmux`**：

        如果你尚未安装 `tmux`，可以使用包管理器进行安装，如`apt`（对于Debian/Ubuntu）或`brew`（对于macOS）。安装后，可以在终端中运行以下命令启动 `tmux`：

        ```bash
        tmux
        ```

        2. **tmux 的基本概念**：

        - **会话（Session）**：一个会话是 `tmux` 的顶级容器，可以包含多个窗口。
        - **窗口（Window）**：一个窗口包含一个或多个窗格，通常用于运行一个命令或应用程序。
        - **窗格（Pane）**：窗格是窗口的子部分，可以在其中运行命令。

        3. **基本命令**：

        - 创建新会话：
            ```bash
            tmux new-session -s session-name
            ```

        - 列出会话：
            ```bash
            tmux list-sessions
            ```

        - 进入会话：
            ```bash
            tmux attach-session -t session-name
            ```

        - 分离会话（保持会话在后台运行）：
            按下 `Ctrl-b`，然后按下 `d`（即`Ctrl-b d`）。

        - 创建新窗口：
            按下 `Ctrl-b`，然后按下 `c`。

        - 切换窗口：
            按下 `Ctrl-b`，然后按下数字键 `0` 到 `9` 来切换到相应编号的窗口。

        - 分割窗格（水平分割）：
            按下 `Ctrl-b`，然后按下 `%`。

        - 分割窗格（垂直分割）：
            按下 `Ctrl-b`，然后按下 `"`。

        - 切换焦点到下一个窗格：
            按下 `Ctrl-b`，然后按下 `o`。

        - 关闭当前窗格：
            按下 `Ctrl-b`，然后按下 `x`。

        4. **其他常用命令**：

        - 重命名窗口：
            按下 `Ctrl-b`，然后按下 `,`。

        - 切换到上一个窗口：
            按下 `Ctrl-b`，然后按下 `l`。

        - 显示当前时间：
            按下 `Ctrl-b`，然后按下 `t`。

        - 查看帮助：
            按下 `Ctrl-b`，然后按下 `?`。

        这些是 `tmux` 的一些基本用法。你可以根据需要进一步探索更多功能和选项，以满足你的特定需求。要退出 `tmux` 会话，可以在其中运行 `exit` 命令或者按下 `Ctrl-b`，然后按下 `:` 来进入命令模式，然后输入 `kill-session` 并按下回车键。    


- vim / nano：文本编辑器。vim 有着丰富的功能和强大的编辑能力，但需要一些学习；nano 则相对简单易用。
    - ### 快捷参考
        Vim 是一个强大的文本编辑器，广泛用于终端环境。它有很多功能和快捷键，学习曲线可能比较陡峭，但一旦掌握，可以显著提高编辑效率。以下是 Vim 的基本使用方法：

        1. **启动 Vim**：

        在终端中输入以下命令以启动 Vim：

        ```bash
        vim
        ```

        或者，如果要编辑一个文件，可以在启动时指定文件名：

        ```bash
        vim 文件名
        ```

        2. **基本编辑**：

        - 切换到插入模式（Insert Mode）：按下 `i` 键。
        - 切换到追加模式（Append Mode）：按下 `a` 键。
        - 从当前光标位置追加一行：按下 `o` 键（小写字母 "o"）。
        - 从当前光标位置向上插入一行：按下 `O` 键（大写字母 "O"）。
        - 退出插入模式并返回普通模式（Normal Mode）：按下 `Esc` 键。

        3. **保存和退出**：

        - 保存当前文件：在普通模式下，按下 `:` 进入命令行模式，然后输入 `w` 并按下回车键。
        - 保存并退出：在普通模式下，按下 `:` 进入命令行模式，然后输入 `wq` 并按下回车键。
        - 不保存并退出：在普通模式下，按下 `:` 进入命令行模式，然后输入 `q!` 并按下回车键。

        4. **导航**：

        - 在普通模式下，使用箭头键或 `h`（左）、`j`（下）、`k`（上）、`l`（右）进行光标导航。
        - 向前跳转一个单词：按下 `w`。
        - 向后跳转一个单词：按下 `b`。
        - 移动到行首：按下 `0`。
        - 移动到行尾：按下 `$`。

        5. **复制、剪切和粘贴**：

        - 复制文本：在普通模式下，按下 `v` 进入可视模式，然后使用箭头键或 `h`、`j`、`k`、`l` 选择文本，再按下 `y` 复制选定文本。
        - 剪切文本：在普通模式下，按下 `v` 进入可视模式，选择文本，然后按下 `x` 剪切选定文本。
        - 粘贴文本：在普通模式下，按下 `p` 将剪切或复制的文本粘贴到光标位置的后面，按下 `P` 将其粘贴到光标位置的前面。

        6. **撤销和重做**：

        - 撤销：在普通模式下，按下 `u`。
        - 重做：在普通模式下，按下 `Ctrl-r`。

        7. **搜索和替换**：

        - 搜索：在普通模式下，按下 `/` 进入搜索模式，然后输入要搜索的文本并按下回车键。可以使用 `n`（下一个匹配项）和 `N`（上一个匹配项）浏览匹配项。
        - 替换：在普通模式下，按下 `:` 进入命令行模式，然后输入 `s/要替换的文本/替换后的文本/g` 并按下回车键，其中 `g` 表示全局替换。

        8. **帮助**：

        在普通模式下，按下 `:` 进入命令行模式，然后输入 `help` 并按下回车键来打开 Vim 的帮助文档。

        这些是 Vim 的基本使用方法，但 Vim 还有许多高级功能和快捷键，可以根据需要进一步学习。要了解更多信息，请查阅 Vim 的在线文档或使用内置的帮助系统。

    ```bash
    brew install fd
    ```

        ```bash
        fd 关键字
        ```

        ```bash
        fd 关键字 /path/to/directory
        ```

        ```bash
        fd -r 关键字
        ```

        ```bash
        fd -e txt
        ```

        ```bash
        fd example
        ```

- ripgrep (rg)：一个非常快速的文本搜索工具。
    ### 用法
    `ripgrep`（简称为`rg`）是一个快速、递归搜索文本的命令行工具，用于在文件和目录中查找匹配的文本。它比传统的`grep`命令更快，同时提供了许多实用的选项。以下是`ripgrep`的基本用法：

    1. **安装 `ripgrep`**：

    `ripgrep` 可以通过包管理器来安装，具体的安装命令可能因你的操作系统而异。以下是一些示例：

    - 在 Ubuntu/Debian 上安装：

        ```bash
        sudo apt-get install ripgrep
        ```

    - 在 Fedora 上安装：

        ```bash
        sudo dnf install ripgrep
        ```

    - 在 macOS 上使用 Homebrew 安装：

        ```bash
        brew install ripgrep
        ```

    - 在 Windows 上使用 Chocolatey 安装：

        ```bash
        choco install ripgrep
        ```

    2. **基本用法**：

    `ripgrep` 的最基本用法是搜索指定的文本模式。以下是示例：

    ```bash
    rg 搜索词
    ```

    这将在当前目录及其子目录中递归地搜索包含 "搜索词" 的文本。

    3. **搜索指定目录**：

    如果你想要在特定目录中搜索，可以指定目录路径作为参数：

    ```bash
    rg 搜索词 /path/to/directory
    ```

    4. **忽略大小写**：

    默认情况下，`ripgrep` 区分大小写。要忽略大小写，可以使用 `-i` 或 `--ignore-case` 选项：

    ```bash
    rg -i 搜索词
    ```

    5. **显示行号**：

    默认情况下，`ripgrep` 会显示匹配的行以及文件名。要显示行号，可以使用 `-n` 或 `--line-number` 选项：

    ```bash
    rg -n 搜索词
    ```

    6. **递归搜索**：

    默认情况下，`ripgrep` 会递归地搜索子目录。如果你只想在当前目录中搜索，可以使用 `-n` 或 `--no-ignore` 选项：

    ```bash
    rg -n 搜索词
    ```

    7. **排除文件和目录**：

    使用 `-g` 或 `--glob` 选项可以排除特定文件或目录：

    ```bash
    rg -g '!*.log' 搜索词  # 排除所有以 .log 结尾的文件
    ```

    8. **正则表达式搜索**：

    你可以使用正则表达式来进行更高级的搜索：

    ```bash
    rg '正则表达式'
    ```

    9. **输出格式**：

    默认情况下，`ripgrep` 会将匹配的行以彩色文本的形式显示。你可以使用 `--color` 和 `--no-color` 选项来自定义颜色输出。

    10. **查看帮助**：

        如果需要了解更多选项和用法，可以运行以下命令来查看 `ripgrep` 的帮助文档：

        ```bash
        rg --help
        ```

    `ripgrep` 是一个非常强大且快速的文本搜索工具，适用于查找文件和目录中的文本。通过上述基本用法，你可以开始使用它，并根据需要深入学习更多高级功能。

        ```bash
        bat 文件名
        ```
